# HDU-Credit-Statistic
杭州电子科技大学学分统计工具，通过爬取教务系统中的各学期成绩与培养计划进行比较，得出学分获得情况。目前还比较简陋，而且效率相当低，登录后需等待约 1 分钟才能出结果。
